# 真涨薪微信小程序

把当前工作与一至两个 Offer 的年综合价值、综合时薪、五险一金和关键反超条件放在同一张账上。计算完全在本机完成，不需要登录，也不上传薪资。

支持通过微信右上角菜单或结果页“分享给朋友”按钮分享小程序入口。分享内容不携带薪资和计算结果，好友进入后自行填写条件。

## 本地验证

```bash
npm run verify
```

## 微信开发者工具导入

1. 打开微信开发者工具，选择“导入项目”。
2. 项目目录选择本目录。
3. 当前 `project.config.json` 已配置小程序 AppID `wxd183b2cabea63847`；如导入其他账号的小程序，请先替换为对应 AppID。
4. 编译后从“当前工作”开始填入数据，依次进入 Offer 与对比结果。

## 计算口径

- 每年工作 48 周。
- 每年通勤 240 天，按往返计算。
- 五险缴费基数、公积金缴存基数和公司公积金比例为可选项。
- 公司公积金年缴存 = 公积金缴存基数 × 公司比例 × 12，并计入年综合价值。
- 税前保证收入不含个税、个人五险一金扣款、公司公积金、试用期折扣、浮动奖金、股票和期权。
- 年综合价值 = 税前保证年收入 + 公司公积金年缴存。
- 综合时薪 = 年综合价值 ÷（年度工作时间 + 年度通勤时间）。

## 项目结构

- `pages/calculator`：输入、校验、本地草稿与示例数据。
- `pages/result`：年综合价值、税前保证收入、综合时薪、年度占用和通勤的深色对比结果。
- `pages/breakeven`：覆盖全部两两工作组合；拖动奖金月数或每周工时后实时试算，不回写原始数据。
- `utils`：与界面无关的计算、校验、格式化和本地存储逻辑。
- `tests`：计算、边界测试，以及“填示例 → 计算 → 结果 → 反超试算”页面流程测试。
