# 真涨薪微信小程序

把当前工作与一至两个 Offer 的税前保证年收入、税前有效时薪和关键反超条件放在同一张账上。计算完全在本机完成，不需要登录，也不上传薪资。

## 本地验证

```bash
npm run verify
```

## 微信开发者工具导入

1. 打开微信开发者工具，选择“导入项目”。
2. 项目目录选择本目录。
3. 当前 `project.config.json` 已配置小程序 AppID `wxd183b2cabea63847`；如导入其他账号的小程序，请先替换为对应 AppID。
4. 编译后从“当前工作”开始填入数据，依次进入 Offer 与对比结果。

## 计算口径

- 每年工作 48 周。
- 每年通勤 240 天，按往返计算。
- 只计算税前保证收入；不含个税、五险一金、试用期折扣、浮动奖金、股票和期权。
- 有效时薪 = 税前保证年收入 ÷（年度工作时间 + 年度通勤时间）。

## 项目结构

- `pages/calculator`：输入、校验、本地草稿与示例数据。
- `pages/result`：年收入、有效时薪、年度占用和通勤的深色对比结果。
- `pages/breakeven`：覆盖全部两两工作组合；拖动奖金月数或每周工时后实时试算，不回写原始数据。
- `utils`：与界面无关的计算、校验、格式化和本地存储逻辑。
- `tests`：计算、边界测试，以及“填示例 → 计算 → 结果 → 反超试算”页面流程测试。
