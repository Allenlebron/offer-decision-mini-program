# 真涨薪微信小程序

把当前工作与一至两个 Offer 的税前保证年收入、税前有效时薪和关键反超条件放在同一张账上。计算完全在本机完成，不需要登录，也不上传薪资。

## 本地验证

```bash
npm run verify
```

## 微信开发者工具导入

1. 打开微信开发者工具，选择“导入项目”。
2. 项目目录选择本目录。
3. 首次体验可使用 `touristappid` 在开发者工具模拟器中运行；生成预览二维码或正式上传前，在 `project.config.json` 中替换为自己的小程序 AppID。
4. 编译后从“输入条件”页填入数据，点击“算清真实涨幅”。

## 计算口径

- 每年工作 48 周。
- 每年通勤 240 天，按往返计算。
- 只计算税前保证收入；不含个税、五险一金、试用期折扣、浮动奖金、股票和期权。
- 有效时薪 = 税前保证年收入 ÷（年度工作时间 + 年度通勤时间）。

## 项目结构

- `pages/calculator`：输入、校验、本地草稿与示例数据。
- `pages/result`：结果对比、反超条件、分享与匿名验证回执。
- `utils`：与界面无关的计算、校验、格式化和本地存储逻辑。
- `tests`：与网站口径一致的计算、边界测试，以及“填示例 → 计算 → 结果 → 反馈回执”页面流程测试。
